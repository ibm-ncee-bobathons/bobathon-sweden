# Full SDLC Overview — What Bob Does, and Why It Matters

## What You'll Experience

This comprehensive demo walks through the complete software delivery lifecycle using IBM Bob, from building a payment processing application in a single prompt, through security scanning, CI/CD pipeline generation, and OpenShift deployment, to the Day-2 SRE operations that drive daily operational toil.

### Key Highlights

- **One Prompt → Full Application**: Watch Bob build a complete payment processing app with backend, frontend, and all components from a single detailed specification
- **Intelligent Code Generation**: See how Bob drives every stage intelligently, generating production-ready artifacts
- **Complete SDLC Coverage**: Experience the full journey from code to production
- **Real-World Scenarios**: Learn through practical examples including Java modernization, security scanning, and infrastructure as code
- **Day-2 Operations**: Understand how Bob handles ongoing operational tasks
